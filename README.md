# Portfolio 2026 - Jérémy Patapy

Portfolio personnel développé avec Next.js 16, TypeScript et Tailwind CSS.

## 🚀 Démarrage rapide

```bash
npm install
npm run dev
```

## 📦 Stack Technique
- Next.js 16
- TypeScript
- Tailwind CSS 4
- Framer Motion (animations)
- next-themes (mode sombre)

## 📁 Structure du projet
- `src/app/` - Pages et layouts
- `src/components/` - Composants réutilisables
- `src/data/` - Données statiques
- `src/css/` - Styles globaux
- `public/` - Assets statiques
